# Data Structures and Algorithms in Java
This project contains custom implementations of core data structures and algorithms, including a hash table, graph, and optimized heap sort. 

---

## Features
### Hash Table
- Stores 10-digit ISBN keys and associated book names.
- Uses a custom hashing function.
- Handles collisions with linear probing.
- Automatically expands capacity when the load factor reaches 70%.
- Supports insertion, search, update, and deletion.
- Validates ISBN input format.

### Graph
- Adds and removes vertices and edges.
- Displays the graph using an adjacency matrix.
- Checks the connectivity of the graph.
- Performs breadth-first and depth-first searches.
- Finds the shortest path between two vertices.

### Optimized Heap Sort
- Implements an optimized heap sorting algorithm using a minheap.
- Builds the heap in place.
- Sorts arrays in descending order.
- Uses Java generics with Comparable<T>
- Handles edge cases such as empty arrays and arrays containing only null values. 

---

## Key Concepts Demonstrated
- Hashing and collision resolution with linear probing.
- Graph traversal (breadth-first and depth-first).
- Shortest path algorithms.
- Heap data structure and heap sorting.
- Generics.
